My Work

  My professional work is dedicated to application of computer algorithms and numerical techniques. Since 1994 I have been doing research work on optimisation of forming processes in C3M , where I have coordinated work on optimisation. Within this scope I have developed the optimisation software Inverse. You can find some information on my work in C3M here.

Software:
  • Optimisatioin shell Inverse, a software for utilisation of various simulatio environments for solution of inverse and optimisation problems; developed within C3M.
  • Optimization library Ioptlib (Investigative optimization library) - a library for development of optimization algorithms, coded in ANSI C.
  • SInt - A fast versatile interpreter, especially suitable for controlling numerical, for interpreting application initialisation scripts, supporting user routines, etc. conception stage only.
  • ITK - a toolkit based on Tcl/Tk, provides extensions with a few useful utilities
  • IGS - a simple graphic system for drawing functions and 3D objects
  • LAF - program for linear approximations, coded in Pascal.
