Igor G. - Software


This page contains links to software based on the legacy Investigative Generic Library (IGLib), developed by Igor Grešovnik and his colleagues. This software is now called IGLib Framework. It is a separate and distinct software from the revised IGLib.NET (or simply IGLib), and is not actively developed.

Legacy Investigative Generic Library (IGLib Framework). Base library for development of technical and other applications.

IGShell - example shell application based on IGLib. Includes a Visual Studio / Mono project and two hosted applications:
  • HashForm - a GUI-based user friendly application for file verification. Performs calculation and verification of various kinds of file or text hash values.
  • HashShell - a command-line application for file verification.

NeurApp - approximation with artificial neural networks (for education & exploration purposes).

AnnApp - building and exploring deep learning regression models (for industrial applications)

Investigative Opimization Library (IOptLib) - an engineering library for continuous optimization, integrated with the IGLib Framework.

Software for modeling linear & nonlinear oscillators by Marko Petek.
